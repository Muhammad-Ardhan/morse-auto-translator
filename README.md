# 📡 Automatic Morse Code Translator

Welcome to my Morse Code Translator project! This is a smart Command Line Interface (CLI) application that automatically translates English text to Morse code and vice versa without requiring manual mode selection.

## 🚀 Quick Start

Want to try it without setup? Click below to launch a live terminal environment in your browser:

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/Muhammad-Ardhan/morse-auto-translator?quickstart=1)

## ✨ Features

- **Auto-Detect System:** Automatically distinguishes between plain text and Morse code symbols (`.` and `-`).
- **Interactive CLI:** Simple loop interface that lets you translate multiple messages continuously.
- **Clean Output:** Clear formatting and status messages for a smooth terminal experience.
