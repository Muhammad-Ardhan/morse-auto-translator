# 📡 Automatic Morse Code Translator

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/Muhammad-Ardhan/morse-auto-translator?quickstart=1)


Welcome to my first official Git hub Upload! This is a smart Command Line Interface (CLI) application that automatically translates English text to Morse code and vice versa without requiring manual mode selection.

## ✨ Features
- **Auto-Detect System:** Automatically distinguishes between plain text and Morse code symbols (`.` and `-`).
- **Bi-directional Translation:** Translates English to Morse or Morse back to English instantly.
- **Input Sanitization:** Embedded with `.strip()` to clean up accidental edge spaces.
- **Optimized Performance:** Dictionary swapping is handled efficiently outside the main loop.

## 🚀 How to Run
1. Copy the code into any Python environment (like Pydroid 3 on mobile or VS Code on PC).
2. Run the script.
3. Type your text or Morse code and press Enter!
